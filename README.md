# 👩‍💼 Employee Data Trend Analysis Dashboard

## 📌 Overview
This project presents a comprehensive **employee trend and compensation analysis** using an interactive dashboard.  
It highlights salary distributions, gender-based trends, departmental performance, and job title bonuses, with a focus on employee data from China.  

The dashboard is designed to help HR teams, managers, and analysts understand workforce dynamics and make informed decisions about compensation, hiring, and performance management.

## 📈 Insights
- **Gender Gap:** Female employees collectively earn slightly more than male employees.  
- **Peak Salary Year:** 2021 shows the highest salary growth for both genders.  
- **Department Highlight:** Marketing leads in average salary performance.  
- **Business Unit Comparison:** Manufacturing edges out Corporate in average salary.  
- **Bonus Leaders:** Senior leadership roles (Vice President, Director) receive the highest bonuses.
